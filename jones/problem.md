# Greedy Two-Layer Neural Network Approximation

This problem implements the iterative greedy approximation algorithm in Jones (1992). The goal is to construct a two-layer neural network:

$$y(x) = \sum_{i=1}^n c_i \sigma(w_i x + b_i), \quad x \in [-1,1]$$

to approximate a target function $f(x)$ through an iterative construction where the $L^2$ error decays as $\mathcal{O}(n^{-1/2})$.

---

## Algorithm Formulation

Let $h_0(x) = 0$ represent the initial approximation. The greedy iterations are computed as follows:

1. **Find the next unit parameter** $(w_k, b_k) \in \Omega$ and scaling factor $\alpha_k \in [0, 1]$:
   
   $$(\alpha_k, w_k, b_k) := \arg\min_{\alpha \in [0, 1], (w, b) \in \Omega} \left\| f(x) - \left( (1-\alpha) h_k(x) + \alpha \sigma(w \cdot x + b) \right) \right\|_{L^2(X)}$$

2. **Update the approximation**:
   
   $$h_{k+1}(x) = (1-\alpha_k) h_k(x) + \alpha_k \sigma(w_k x + b_k)$$

### Key Implementation Details

1. **Optimal $\alpha_k$ Calculation**:
   Instead of using an iterative solver for $\alpha_k$, we derive the optimal unconstrained scaling factor analytically. Let $A(x) = f(x) - h_k(x)$ and $B(x) = \sigma_k(x) - h_k(x)$. We want to minimize the quadratic $\| A - \alpha B \|^2_{L^2(X)}$. Equating the derivative to zero yields the unconstrained optimal $\alpha^*$:
   
   $$\alpha^* = \frac{\langle A, B \rangle}{\|B\|^2} = \frac{\langle f - h_k, \sigma_k - h_k \rangle}{\|\sigma_k - h_k\|^2}$$
   
   Since $\alpha_k$ must lie in $[0, 1]$ to remain a valid convex combination, we clip $\alpha^*$ to get the optimal constrained value:
   
   $$\alpha_k = \max\left(0, \min\left(1, \alpha^*\right)\right)$$

2. **Sigmoid Selection via Random Sampling**:
   To find $(w_k, b_k)$, the Jones greedy algorithm requires finding a sigmoid $\sigma_k(x) = \sigma(w_k x + b_k)$ satisfying the descent inequality:
   
   $$\langle f(x) - h_k(x), f(x) - \sigma(w_k x + b) \rangle < 0$$
   
   We implement this by sampling random linear layers $w \cdot x + b$ where $w, b \sim \mathcal{U}(-10, 10)$ until the inequality holds (up to 5000 trials, falling back to the best candidate if none satisfy it).
